			LPJ-GUESS-HYD Amazon modelling study
                        ===================================

                        PLEASE READ CAREFULLY
 
This directory contains scripts for post processing and other files necessary to
create the figures of the Amazon modelling study.

It does not contain the source code and the model output files of this study. 

Access to the output files is possible by contacting Phillip Papastefanou (pp.papastefanou@gmail.com). 

The source code can be accessed by contacting Johan Nord from the LPJ-GUESS community following https://web.nateko.lu.se/lpj-guess/contact.html
